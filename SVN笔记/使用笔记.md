# SVN 使用笔记（常用命令）

## 1. 仓库基础

```bash
svn checkout <repo_url> <path>
svn co <repo_url> <path>

svn status
svn add <file>
svn add --force .                   # 递归添加当前目录新增文件/目录
svn commit -m "xxx"
svn commit . -m "xxx"               # 提交当前目录
```

## 2. 差异与日志

```bash
svn diff
svn diff <file>

svn log
svn log -l 10
svn info
svn blame <file>
```

## 3. 分支

```bash
svn copy <trunk_url> <branch_url> -m "create branch"
svn switch <branch_url>

svn merge <branch_url>

svn delete <branch_url> -m "delete branch"
```

## 4. 远端与同步

```bash
svn update
svn up

svn commit -m "xxx"
svn cleanup
```

## 5. 撤销/回滚（高频）

```bash
svn revert <file>                   # 丢弃工作区修改
svn revert -R .                     # 递归丢弃全部工作区修改

svn merge -c -<revision> <repo_url> # 反向合并指定版本
svn commit -m "revert r<revision>"
```

## 6. 暂存区与提交小技巧

```bash
svn status
svn diff
svn commit <file> -m "xxx"          # 只提交指定文件
svn commit --targets <file_list> -m "xxx"
```

## 7. 忽略文件

```bash
svn propset svn:ignore "<pattern>" <dir>
svn propedit svn:ignore <dir>
svn propget svn:ignore <dir>
```

## 8. Tag（打版本）

```bash
svn copy <trunk_url> <tag_url> -m "v1.0.0"
svn list <tags_url>
svn delete <tag_url> -m "delete tag"
```

## 9. 文件与目录操作

```bash
svn mkdir <dir>
svn move <old> <new>
svn copy <src> <dst>
svn delete <path>
```

## 10. 清理（谨慎）

```bash
svn cleanup                         # 清理锁/中断状态
svn cleanup --remove-unversioned    # 删除未版本控制文件
svn cleanup --remove-ignored        # 删除忽略文件
```
