# 大富翁游戏

## 项目介绍
这是一个基于网页的大富翁游戏，玩家可以在游戏中进行掷骰子、购买地产、收取租金等操作。游戏的目标是通过策略和运气使自己成为最后的赢家。

## 文件结构
```
monopoly-game
├── src
│   ├── index.html       # 游戏的主页面
│   ├── styles
│   │   └── style.css    # 游戏的样式定义
│   └── scripts
│       └── app.js       # 游戏的主要逻辑实现
└── README.md            # 项目的文档
```

## 安装步骤
1. 克隆该项目到本地：
   ```
   git clone <项目地址>
   ```
2. 进入项目目录：
   ```
   cd monopoly-game
   ```
3. 打开 `src/index.html` 文件在浏览器中查看游戏。

## 使用说明
- 点击“掷骰子”按钮开始游戏。
- 根据骰子的点数移动你的棋子。
- 购买地产并收取租金。
- 通过策略和运气成为最后的赢家！
