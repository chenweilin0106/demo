# 咸鱼之王词条关系说明

记录时间：2026-06-15

用途：按“关系组”整理词条，方便直接看相互抵消、上限和高低效果。后续新增词条时，优先新增关系组；单个词条只放到文末索引。

## 关系总览

| 关系组 | 进攻/触发词条 | 防守/抵抗词条 | 核心结论 |
|---|---|---|---|
| 破甲关系 | 破甲 | 破抵、破甲抵抗 | 破甲和破抵相减，只剩差额生效 |
| 控制关系 | 控制命中、控制概率 | 免控、指定免疫 | 控制命中抵消免控；指定免疫不被控制命中抵消 |

## 1. 破甲关系：破甲 vs 破抵

### 相关词条

| 词条 | 含义 |
|---|---|
| 破甲 | 按比例无视目标的防御减伤 |
| 破抵 | 破甲抵抗，用来抵消攻击方破甲 |
| 实战破甲 | 破甲和破抵抵消后，真正参与伤害计算的破甲 |

### 关系公式

```text
实战破甲 = clamp(攻击方破甲 - 防守方破抵, 0%, 70%)
实战防御减伤 = 原防御减伤 - 实战破甲
```

说明：
- `破抵` 只抵消 `破甲`，不倒扣攻击方伤害。
- `实战破甲` 通常按 `70%` 上限理解，因为防御减伤上限是 `70%`。
- PVE 或特殊目标可能自带破抵，具体以目标机制为准。

### 后期 PVP 简化口径

后期武将防御值通常已经顶到 `70%` 防御减伤，可直接简化为：

```text
最终防御减伤 = max(70% - (攻击方破甲 - 防守方破抵), 0)
最终伤害比例 = 1 - 最终防御减伤
```

例子：
- `破甲 - 破抵 = 0%`：最终防御减伤 `70%`，只打出 `30%` 伤害。
- `破甲 - 破抵 = 40%`：最终防御减伤 `30%`，打出 `70%` 伤害。
- `破甲 - 破抵 >= 70%`：最终防御减伤 `0%`，防御被打穿。

### 高低效果

| 对比 | 效果 |
|---|---|
| 破甲 > 破抵 | 差额成为实战破甲，提高伤害 |
| 破甲 = 破抵 | 实战破甲为 `0%` |
| 破甲 < 破抵 | 实战破甲仍为 `0%`，多出的破抵不额外减伤 |
| 破甲 - 破抵 > 70% | 超出 `70%` 的部分通常没有收益 |

### 例子

| 攻击方破甲 | 防守方破抵 | 实战破甲 | 结果 |
|---:|---:|---:|---|
| 100% | 40% | 60% | 破甲仍有效 |
| 100% | 100% | 0% | 破甲被抵消 |
| 100% | 130% | 0% | 多出的 30% 破抵不倒扣 |
| 180% | 80% | 70% | 差额 100%，但通常只按 70% 生效 |

## 2. 控制关系：控制命中 vs 免控 vs 免疫

### 相关词条

| 词条 | 含义 |
|---|---|
| 控制概率 | 技能原本写的眩晕、冰冻、沉默概率 |
| 控制命中 | 用来抵消目标免控；有些口径下控制命中高于免控时会放大实战控制概率 |
| 免控 | 被控制时，免疫这次控制的概率 |
| 指定免疫 | 针对单一状态的抗性，如眩晕免疫、冰冻免疫、沉默免疫、灼烧免疫、流血免疫、中毒免疫 |

### 关系公式

控制命中和免控：

```text
控制修正 = 1 + 攻击方控制命中 - 防守方免控
实战控制概率 = 控制概率 × 控制修正
```

保守使用口径：
- `控制命中` 主要用于抵消 `免控`。
- 若 `控制命中 > 免控`，按常见攻略口径，实战控制概率会被放大。
- 若采用“控制命中不能超过原控制概率”的口径，则只把控制修正上限看作 `1`。

指定免疫：

```text
实战被挂状态概率 = 状态概率 × (1 - 对应免疫)
```

说明：
- `免控` 只管眩晕、冰冻、沉默。
- `免控` 不影响灼烧、流血、中毒。
- `控制命中` 不抵消指定免疫。
- 指定免疫只管对应状态，例如 `冰冻免疫` 不管眩晕。

### 高低效果

| 对比 | 效果 |
|---|---|
| 控制命中 > 免控 | 免控被抵消；按常见口径，控制概率可能提高 |
| 控制命中 = 免控 | 按技能原控制概率判断 |
| 控制命中 < 免控 | 剩余免控降低控制成功率 |
| 指定免疫 > 0 | 直接降低对应状态挂上概率 |
| 指定免疫 = 100% | 通常不会被对应状态挂上 |

### 例子

| 场景 | 计算 | 结果 |
|---|---|---|
| 20% 眩晕，50% 控制命中，0% 免控 | `20% × (1 + 50% - 0%)` | `30%` 眩晕 |
| 75% 眩晕，20% 控制命中，60% 免控 | `75% × (1 + 20% - 60%)` | `45%` 眩晕 |
| 75% 眩晕，0% 控制命中，40% 眩晕免疫 | `75% × (1 - 40%)` | `45%` 眩晕 |
| 100% 灼烧，60% 免控 | 免控不参与 | 仍按灼烧自身概率和灼烧免疫判断 |
