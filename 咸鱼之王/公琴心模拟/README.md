# 公琴心模拟文档索引

用途：集中记录吕布使用 5 星公琴心时的可复用输入、公式口径、对手假设和模拟结论，避免后续反复从截图或聊天记录中提取信息。

## 文件说明

- `模拟输入记录.md`
  - 记录当前“核爆”阵容、五名上阵武将、鱼灵、玩具、基础属性、特殊属性和淬炼。
  - 后续如果属性变化，优先更新这个文件。

- `机制与公式口径.md`
  - 记录公琴心、月光、吕布、公孙瓒、暴击期望、琴心层数等模拟公式。
  - 后续所有模拟都按这里的公式走，除非明确改口径。

- `对手常规阵容假设.md`
  - 记录吴国、姜维、典韦三类对手的常规阵容假设。
  - 这些是假设模板，不是具体玩家面板。

- `公琴心_vs_月光_吕布模拟报告.md`
  - 记录本轮针对三种指定场景的输出对比结果。

## 当前固定模拟前提

- 公琴心：5 星。
- 吕布：按三件套后 `107.6%` 技能伤害模拟。
- 对比对象：只比较 `月光吕布` 与 `公琴心吕布`。
- 输出指标：吕布第二轮大招期望输出相对倍率。
- 不做静默兜底：缺少对手真实面板时，只做假设模板和敏感度分析，不伪装成精确战报。

## 后续更新顺序

1. 更新 `模拟输入记录.md`：你的阵容和面板。
2. 更新 `对手常规阵容假设.md`：对手阵容与关键属性。
3. 检查 `机制与公式口径.md`：是否有新鱼灵、新三件套或机制改动。
4. 生成或更新对应模拟报告。
