# 指定阵容复测结论

记录时间：2026-05-19

固定前提：
- 公琴心：5 星。
- 吕布：按三件套后 `107.6%` 技能伤害。
- 对比：只比较月光和公琴心。
- 指标：吕布第二轮大招期望输出相对倍率。
- 抵抗口径：常规抵抗目标，按 `30%` 暴击抵抗、`30%` 爆伤抵抗估算。

## 关键澄清

公琴心不是靠三件套才超过月光。按完整鱼灵属性做大招对比，公琴心大招仍然更高：

| 琴心层数 | 纯被动对月光大招倍率 |
|---:|---:|
| 0 | 1.12 倍 |
| 1 | 1.16 倍 |
| 2 | 1.20 倍 |
| 3 | 1.24 倍 |
| 4 | 1.28 倍 |
| 5 | 1.32 倍 |

三件套 `20%` 技能伤害对月光和公琴心都生效，通常不会改变两条鱼之间的相对倍率；它提高的是吕布大招基础盘，所以公琴心乘上去后的绝对伤害更大。

## 指定阵容

吴国：
- 周瑜、孙策、太史慈、曹仁、大乔

姜维：
- 姜维、孙策、曹仁、贾诩、大乔

典韦：
- 典韦、曹仁、甄姬、太史慈、大乔

蜀国多控：
- 赵云、诸葛亮、黄月英为核心
- 其他位按曹仁/大乔/姜维/甄姬等多控补位处理

## 复测结果

| 阵容 | 公孙瓒残血未死，第二轮开大 | 公孙瓒死亡，第二轮开大 | 两层献祭 + 心魔，第二回合 |
|---|---:|---:|---:|
| 吴国 | 1.349 倍 | 1.519 倍 | 1.610 倍 |
| 姜维 | 1.349 倍 | 1.519 倍 | 1.610 倍 |
| 典韦 | 1.349 倍 | 1.519 倍 | 1.610 倍 |
| 蜀国多控 | 1.307 倍 | 1.472 倍 | 1.610 倍 |

说明：
- 表格倍率是在“纯被动已高于月光”的基础上，再加入公琴心 `30%` 暴击、吕布乱世/心魔稀释月光攻击优势后的期望倍率。
- 吴国、姜维、典韦三套指定阵容，在输出倍率上没有本质差异，因为公琴心与月光的差异主要来自吕布自身状态和琴心层数。
- 真正改变选择的是“开大稳定性”，不是单发伤害公式。
- 蜀国多控前两档琴心层数按更保守估算，所以倍率略低；开心魔低血线后仍按 5 层处理。

## 阵容结论

吴国：
- 公琴心优先。
- 吴国血线和护盾使琴心层数较高，控制压力相对低，第二轮核爆收益容易兑现。

姜维：
- 伤害上公琴心优先，稳定性上月光更稳。
- 这套姜维带贾诩和大乔，没有甄姬解控；如果我方郭嘉能控住关键点，公琴心收益很好。
- 如果吕布经常被大乔沉默、曹仁冰冻或贾诩干扰到开不出大，月光优先。

典韦：
- 公琴心优先用于抢窗口。
- 典韦、曹仁、甄姬、太史慈、大乔这套越拖越强，公琴心适合第二轮压死/压残。
- 如果第二轮压不穿，后面月光更适合拖心魔吸血循环。

蜀国多控：
- 默认月光更稳。
- 有璇玑/冰清郭嘉应对时，可以把公琴心作为进攻选择；前提是郭嘉能稳定限制对方控制链。
- 若吕布第二轮开大率不稳定，公琴心理论伤害再高也不如月光。

## 最终选择

- 吴国：公琴心。
- 姜维：能开大用公琴心，开不出大用月光。
- 典韦：抢第二轮用公琴心，拖后续用月光。
- 蜀国多控：默认月光；郭嘉控制链稳定时可试公琴心。
