## 固定输入

公琴心：
- 星级：5 星
- 暴击：`+30%`
- 减伤：`+30%`
- 破甲：`+100%`
- 释放技能时，造成的技能伤害提升：`40%`
- 每有 1 名敌方当前生命百分比不低于携带者，本次技能伤害额外提升：`5%`
- 多条琴心同时存在时，仅生效 1 条

月光：
- 攻击：`+25%`
- 免控：`+25%`
- 破甲：`+100%`
- 另有满月盾和负面持续缩短，当前输出模型不把护盾折算成伤害。

吕布三件套：
- 额外技能伤害：`+20%`
- 当前截图吕布技能伤害：`87.6%`
- 三件套后吕布技能伤害：`107.6%`

## 公琴心层数

层数定义：

```text
琴心层数 = 释放技能时，敌方当前生命百分比 >= 携带者当前生命百分比 的敌方人数
```

取值范围：

```text
0 到 5
```

公琴心技能倍率：

```text
琴心技能倍率 = 1.40 + 0.05 × 琴心层数
```

例子：
- 0 层：`1.40`
- 2 层：`1.50`
- 5 层：`1.65`

## 公琴心与月光输出对比

本模拟只比较吕布大招期望伤害，不比较完整胜率。

核心公式：

```text
公琴心 / 月光
= 攻击修正 × 暴击期望修正 × 琴心技能倍率
```

因为公琴心和月光都提供 `100%` 破甲，破甲在两者对比中抵消。

基础被动对比：

```text
不计暴击差异、不计乱世/心魔：
公琴心 / 月光 = (1.40 + 0.05 × 琴心层数) / 1.25
```

| 琴心层数 | 只看被动后的公琴心 / 月光 |
|---:|---:|
| 0 | 1.12 |
| 1 | 1.16 |
| 2 | 1.20 |
| 3 | 1.24 |
| 4 | 1.28 |
| 5 | 1.32 |

结论：只看技能单发伤害，按完整鱼灵属性对比，公琴心被动本身仍然高于月光；三件套技能伤害不会让相对倍率额外变大，它主要增加绝对伤害。

## 攻击修正

保守口径：
- 月光的 `25%` 攻击按完整优势处理。
- 公琴心没有攻击属性。

若吕布已有乱世/心魔攻击加成：

```text
攻击修正 = (100% + 吕布额外攻击加成) / (125% + 吕布额外攻击加成)
```

常用场景：

| 场景 | 吕布额外攻击加成 | 攻击修正 |
|---|---:|---:|
| 无献祭、未心魔 | 0% | 0.800 |
| 1 层乱世 | 30% | 0.839 |
| 2 层乱世 | 60% | 0.865 |
| 心魔，无乱世 | 150% | 0.909 |
| 2 层乱世 + 心魔 | 210% | 0.925 |

说明：如果游戏内攻击加成不是完整独立相乘，而是进入更大的攻击加成池，那么月光 `25%` 攻击的实际边际会更低，公琴心相对收益会高于本口径。

## 暴击期望

暴击期望公式：

```text
有效暴击率 = clamp(我方暴击 - 敌方暴击抵抗, 0%, 100%)
有效爆伤增幅 = max(0, 50% + 我方爆伤 - 敌方爆伤抵抗)
暴击期望 = 1 + 有效暴击率 × 有效爆伤增幅
```

本模型默认：
- 基础暴击倍率为 `1.5`。
- 吕布截图面板爆伤为 `0%`。
- 吕布乱世会叠爆伤。

吕布觉醒乱世：
- 每死亡 1 名友方，吕布增加 `30%` 攻击、`15%` 暴击、`30%` 爆伤。

## 公孙瓒献祭层

公孙瓒觉醒白马将军口径：
- 100%、75%、50%、25%、0% 血线各触发一次。
- 每次为全体友军提供约 `6%` 增伤和 `6%` 减伤。

常用简化：

| 公孙瓒状态 | 已触发血线 | 队伍增伤/减伤 |
|---|---:|---:|
| 满血站场 | 100% | 6% |
| 残血未死 | 100/75/50/25 | 24% |
| 已死亡 | 100/75/50/25/0 | 30% |

说明：本轮公琴心 vs 月光只比较同一阵容下换鱼，公孙瓒提供的增伤减伤通常对两边同时生效，因此不进入相对倍率差异；它只影响“场景描述”和吕布是否获得乱世层。

## 三种本轮场景

场景 A：公孙瓒残血未死，第二轮开大
- 吕布乱世层：`0`
- 心魔：否
- 默认琴心层数：`4`

场景 B：公孙瓒死亡，第二轮开大
- 吕布乱世层：`1`
- 心魔：否
- 默认琴心层数：`5`

场景 C：两层献祭 + 开心魔，第二回合
- 吕布乱世层：`2`
- 心魔：是
- 默认琴心层数：`5`

## 明确不纳入当前输出模型的因素

以下因素会影响胜率，但当前不折算成单发大招伤害：
- 月光满月盾。
- 月光负面持续缩短。
- 控制、降怒、冰冻、沉默导致吕布无法开大。
- 吕布大招随机 3 目标的命中分布。
- 格挡与精准。
- 对方护盾、复活、解控、免死或特殊减伤。
- 太史慈实际出手次数与增伤层数。

这些因素要单独做“完整回合模拟”或“胜率风险评估”，不能混进单发输出倍率里。
