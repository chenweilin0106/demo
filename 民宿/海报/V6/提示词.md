# V6 HTML 民宿图文海报

## 目标

使用 HTML + CSS 重构 `朝汐里·海边小院` 民宿介绍类竖版长图，并用 `$agent-browser` 截图导出。

## 视觉方向

- 海边石屋旅行画册。
- 冷海蓝、石墙灰、原木暖色。
- 沉浸封面、错位画册网格、周边风景拼贴。

## 内容

- 主标题：朝汐里·海边小院
- 副标题：住进海边，推窗见海
- 房型：海景双床房、舒适大床房、海景阳台
- 空间：公共客厅
- 周边：海边廊道、悬崖海岸、海岛渔港、礁石景观

## 约束

- 房型和周边景色使用原图。
- 字体使用 PingFangSC。
- 不用 AI 重绘房间和风景。
- 不写价格、距离、床宽、服务承诺等未确认信息。
- 不限制最大高度，使用 `$agent-browser --full` 导出完整长图。
