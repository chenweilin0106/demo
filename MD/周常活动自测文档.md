## 积分兑奖

- ### 交互

  - [ ] 任务：去完成，跳转充值界面，回到页面会刷新页面数据
  - [x] 任务：领取，领取成功toast，更新按钮状态为已完成，任务次数更新，我的道具数量更新正常；领取失败toast
  - [x] 任务：已完成，点击无效
  - [x] 兑换好礼： 兑换，判断道具数量是否能够兑换一次；数量足够，打开兑换好礼弹框。钥匙特殊处理，只能换两个，未兑换过，判数量需要大于50，兑换第二个，数量需要大于25；数量不足，toast提示：**道具**数量不足（道具名根据具体活动来）
  - [x] 兑换好礼：已兑换，点击无效
  - [x] 兑换好礼弹框-计数器：点击最小，设置为1，点击减号图标无效，并且图标变暗，；点击最大，根据当前道具数量和兑换次数算出最大兑换数量，点击加号图标无效，并且图标变暗；
  - [x] 兑换好礼弹框-兑换：绿晶石，科举卡，紫晶石，召唤石，礼物（魔术帽，OK绷，聚宝盆等）等，兑换成功，toast提示，关闭弹框，道具数量更新正常，兑换次数更新正常；兑换失败，弹框关闭，toast提示；
  - [x] 兑换好礼弹框-称号，头饰，声波，主页特效，铭牌兑换：兑换成功，toast提示，关闭弹框，道具数量更新正常；兑换并装扮，成功，toast提示，关闭弹框，我的道具数量更新正常，兑换次数更新正常，按钮状态更新正常；用户信息更新正常（开发接口，装扮**新装扮**大概率失败，因为后端未上传）；兑换失败，弹框关闭toast提示；

## 充值有礼

- ### 交互

  - [ ] 天天首充礼：点击规则图标，打开规则说明弹框，背景锁定，点击遮盖层关闭；
  - [ ] 天天首充礼：点击立即充值，跳转充值界面，回到页面会刷新页面数据
  - [ ] 天天首充礼：点击去完成，跳转充值界面，回到页面会刷新页面数据；点击领取，请求正常，弹出弹框，大于等于10倍好运爆棚，小于10倍天降好运；请求错误，toast提示；背景锁定，点击遮盖层或者我知道了按钮，关闭弹框
  - [ ] 好运礼盒：点击奖池，弹出奖池说明弹框，背景锁定，点击遮盖层关闭弹框
  - [ ] 好运礼盒：点击抽奖1次/抽奖10次，次数不足，打开次数不足弹框，背景锁定，点击立即前往，关闭弹框，跳转充值界面，回到页面会刷新页面数据；点击遮盖层或者取消按钮，关闭弹框；
  - [ ] 好运礼盒：点击抽奖1次/抽奖10次，请求正常，打开恭喜获得弹框，背景锁定，更新开启次数，点击遮盖层或者我知道了按钮，关闭弹框；请求错误，toast提示
  - [ ] 累计充值有礼，连续充值有礼：点击去完成，跳转充值界面，回到页面会刷新页面数据；点击领取，请求正常，toast提示，更新按钮状态，排序动画正常播放；请求错误，toast提示；已完成，点击无效
