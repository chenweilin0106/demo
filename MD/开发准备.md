1.下载git：https://git-scm.com/downloads

2.设置用户信息

    1. git config --global user.name "Your Name"
    2. git config --global user.email "

3.生成ssh并绑定

    1. ssh-keygen -t ed25519 -C "your_email@example.com"
    2. c:/Users/YOU/.ssh/id_ed25519.pub
    3. 绑定gitlab/github/gitee

4.下载nvm：https://github.com/coreybutler/nvm-windows/releases
    
    1. nvm install "node version"
    2. nvm use "node version"

5.下载pnpm:
    
    1. npm install -g pnpm
    2. 系统禁用脚本 
        1. 先获取脚本执行策略：Get-ExecutionPolicy 应该是 "Restricted"
        2. 设置脚本执行策略：Set-ExecutionPolicy RemoteSigned

6.安装rimraf

    1. npm install rimraf -g
    2. rimraf node_modules

6.clone项目

    1. git clone "url"

7.下载webstorm：

    1.https://www.jetbrains.com/webstorm/
    2.设置内存大小
    3.windows.defender排除webstorm安装目录，项目目录
