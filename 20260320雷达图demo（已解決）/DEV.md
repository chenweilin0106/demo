# 雷达图（SVG）开发文档

## 目标
- 使用 `HTML + CSS + JS` 绘制雷达图，核心使用 `SVG`。
- 画布使用 `viewBox="0 0 800 800"`，页面内自适配缩放显示。
- 维度：5 个维度（能力1~能力5），背景 5 层。
- 每个维度数值范围：`0~5`，仅整数。

## 视觉规范
- 背景五边形线：`#fde7bf`
- 背景点：`#f1ca84`
- 数据五边形线/点：`#de7637`
- 数据填充：`rgba(222, 118, 55, 0.5)`
- 刻度数字：`#835822`，加粗
- 动画时长：`300ms`

## 结构分层（SVG）
建议按从下到上顺序：
1. 背景环（5 层五边形 `path`）
2. 轴线（中心到外圈 `line`）
3. 背景点（每轴每层一个 `circle`）
4. 刻度数字（每轴 1~5 级的 `text`）
5. 维度名称（能力1~能力5 的 `text`）
6. 数据五边形（`path`）
7. 数据点（每轴每层一个 `circle`，用于点亮表现）
8. 数据顶点（5 个 `circle`，用于强调当前值）
9. 中心点（常亮 `circle`）

## 几何算法（正五边形）
- 画布：`size=800`
- 中心：`cx=cy=size/2`
- 最大半径：`rMax=size/2-padding`
- 轴数量：`m=5`
- 起始角度：`-90°`（顶点朝上），每条轴角度间隔 `360/m`
- 极坐标转笛卡尔：
  - `x = cx + r * cos(angle)`
  - `y = cy + r * sin(angle)`

背景第 `k` 层（1..5）半径：
- `r = rMax * (k/5)`

## 数据规则
- 输入值 `v` 属于 `0..5`，对每条轴归一化：`t=v/5`，顶点半径 `r=rMax*t`
- 全 0：不展示数据五边形（透明隐藏），中心点仍展示

### 点亮规则（每轴）
- 设该轴值为 `v`：
  - 刻度点 `1..v` 点亮（`opacity: 1`）
  - `v=0` 时该轴不点亮任何刻度点
- 点亮圆点样式：
  - 点亮：`fill: #de7637; opacity: 1`
  - 未点亮：`opacity: 0`
  - 通过 CSS `transition: opacity 300ms` 过渡

## 刻度数字定位（按轴方向微调）
为避免文字覆盖点位，对不同轴方向施加不同偏移（`dx/dy`）与 `text-anchor`。
示例策略：
- 顶部轴（能力1）：数字在点右侧
- 右上轴（能力2）：数字在点下方
- 右下轴（能力3）：数字在点右侧
- 左下轴（能力4）：数字在点左侧
- 左上轴（能力5）：数字在点下方

## 动画方案
- 使用 `requestAnimationFrame` 在 `300ms` 内插值更新：
  - 数据五边形 `path d`
  - 数据顶点 `circle(cx,cy)`
- 点亮刻度圆点使用 CSS `opacity` 过渡（无需逐帧更新）

## 文件说明
- `index.html`：页面结构（水平/垂直居中）与 SVG 容器
- `styles.css`：颜色、字号、图层样式、透明过渡
- `radar.js`：生成背景/数据层与 `setValues()` API + 简单 demo
