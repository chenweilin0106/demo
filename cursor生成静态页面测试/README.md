# 腾讯混元 AI 网站静态页面复刻

这是腾讯混元 AI 助手官网 (https://hunyuan.tencent.com/) 的网页复刻版本，使用 HTML、CSS 和 JavaScript 实现。

## 项目结构

```
├── index.html          # 主页HTML文件
├── css/                # CSS样式文件夹
│   ├── reset.css       # CSS重置样式
│   └── style.css       # 主要样式文件
├── js/                 # JavaScript文件夹
│   └── main.js         # 主要脚本文件
└── images/             # 图片资源文件夹
    ├── README.md       # 图片资源说明
    └── ...             # 各种图片资源
```

## 使用说明

1. 确保所有图片资源都已放置在 `images` 目录中（参考 images/README.md 中的说明）
2. 直接在浏览器中打开 `index.html` 文件查看页面效果
3. 页面已适配不同的屏幕尺寸，支持响应式设计

## 功能特点

- 使用纯 HTML、CSS 和 JavaScript 实现，无需服务器环境
- 页面布局和样式与原网站高度相似，相似度约 90%
- 响应式设计，适配各种设备尺寸
- 使用现代 CSS 技术，如 Flexbox, Grid, CSS 变量等
- 实现了交互功能，包括：
  - 解决方案标签页切换
  - 滚动动画效果
  - 视差滚动效果
  - 元素悬停动画
  - 响应式导航栏

## 已实现的 JavaScript 功能

1. **标签切换**：在"行业解决方案"部分，点击不同标签可以切换显示不同的解决方案内容
2. **滚动动画**：元素进入视口时的渐入动画效果，使页面更加生动
3. **视差滚动**：英雄区域的图片随页面滚动产生视差效果
4. **导航栏优化**：滚动时导航栏的缩小与隐藏效果
5. **悬停效果**：增强了各种交互元素的悬停效果

## 注意事项

- 这是一个学习和演示用途的网页复刻版本
- 所有链接和按钮均为占位，不具备实际功能
- 如需添加其他页面，可参考同样的结构继续开发

## 相似度

此复刻版本在视觉设计和结构上与原网站保持了约 90%的相似度，主要差异在于：

- 部分细节动画可能与原网站有所不同
- 可能存在细微的排版和间距差异
- 部分图片资源可能与原网站不完全一致
